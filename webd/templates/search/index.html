Transliteration search string

The following wildcard functions can be used:

"*": dummy for any sign, e.g. in "*?m"
"[ ]": set of signs; any of the included character will be searched for, e.g. "s[d?].t"
"?": the preceding sign can occur, but need not occur, e.g. "mrr?.t"
